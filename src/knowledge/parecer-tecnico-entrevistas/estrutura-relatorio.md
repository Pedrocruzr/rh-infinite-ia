# ESTRUTURA DE SAÍDA FINAL

## Formato
Markdown

## Regra principal
O parecer final deve seguir exatamente o modelo compatível com o nível da vaga.

## Regras de redação
- Sempre explicitar evidências
- Não usar linguagem vaga
- Não usar termos discriminatórios
- Não incluir informações pessoais irrelevantes
- Quando uma informação não for fornecida, registrar tecnicamente essa ausência
- Recomendação final deve ser coerente com o corpo do parecer

## Recomendações possíveis
- Aprovado
- Aprovado com Restrições
- Reprovado
