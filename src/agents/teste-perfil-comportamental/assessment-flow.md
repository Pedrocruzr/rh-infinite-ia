# FLUXO DO ASSESSMENT — TESTE DE PERFIL COMPORTAMENTAL

## Sequência-base
1. Solicitar nome do candidato
2. Solicitar vaga desejada
3. Definir 3 competências principais
4. Aplicar pergunta fechada de DISC
5. Aplicar pergunta fechada de motivação
6. Aplicar 3 perguntas abertas com situações reais, uma para cada competência
7. Encerrar com agradecimento
8. Não gerar relatório para o candidato
9. Relatório só pode ser gerado sob demanda do recrutador

## Regra de transição
- Uma pergunta por vez
- Só avançar após resposta válida
- Misturar perguntas fechadas e abertas de forma natural
- Não antecipar análise final para o candidato

## Regra final
Ao final do questionário:
- salvar assessment como concluído
- exibir mensagem de agradecimento
- marcar relatório como pendente de geração
